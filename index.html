Full Stack Basics

This unit covers the basics of full-stack web development. Topics include understanding web development frameworks, Single Page Applications (SPA), and the Model-View-Controller (MVC) architecture. You will also learn about the structure of web pages and HTML tags such as basic tags, links, lists, tables, images, videos, audios, and forms. CSS will be covered for styling, along with Bootstrap for responsive web design elements like containers, tables, buttons, cards, and more.

  • Basic HTML tags, links, and forms.
  • CSS styling and responsive design with Bootstrap.

JavaScript

This unit introduces JavaScript, one of the most essential languages for web development. You will learn about data types, variables, arrays, strings, and operators. The unit covers looping constructs, functions, and dialog boxes, along with event handling. Additionally, you will work with the DOM (Document Object Model), learn about callback functions, promises, arrow functions, and the Fetch API to interact with web resources.

  • Loops, functions, and event handling.
  • DOM manipulation and using the Fetch API.

React

React is a powerful JavaScript library for building user interfaces, particularly for Single Page Applications. In this unit, you'll learn about the Virtual DOM, and how to create a React application using create-react-app. The unit also covers React JSX, component lifecycle methods, hooks, props, and state management. You'll dive into event handling, routing with React Router, and building forms with React. Additionally, you'll learn about conditional rendering and using the Fetch API to populate data in React components.

  • React components, JSX, and hooks.
  • Routing with React Router and form handling.

Node.js & Express.js

This unit provides an introduction to server-side programming using Node.js and Express.js. You will learn how to install Node.js, use the Node Package Manager (NPM), and explore core modules. The unit covers routing in Node.js and how to create simple APIs with Express.js. You'll also get practical experience in building RESTful APIs, managing requests, and handling responses.

  • Routing in Node.js and creating APIs with Express.js.
  • Building RESTful APIs and handling HTTP requests.

MongoDB

MongoDB is a NoSQL database that is widely used in modern web development. In this unit, you will learn about managing collections, performing CRUD (Create, Read, Update, Delete) operations, and building RESTful APIs. You will explore how to connect to MongoDB from Node.js/Express.js and develop simple applications that interact with MongoDB to manage data.

  • Perform CRUD operations with MongoDB.
  • Integrate MongoDB with Node.js/Express.js.
